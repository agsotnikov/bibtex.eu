---
title: Wie kann ich BibTeX verwenden, um eine Website in LaTeX zu zitieren?
---

Es gibt mehrere Möglichkeiten, eine Website in LaTeX zu zitieren. In dieser kurzen Antwort auf die Frage geben wir drei an, die einfachste ist BibLaTeX über die CiteDrive-Browsererweiterung im letzten Beispiel.

:::tip
[CiteDrive](https://www.citedrive.com/) ist ein kollaboratives Online-Referenzmanagement-Tool, das auf BibTeX basiert und sich automatisch mit Publikationssystemen wie [Overleaf](https://www.overleaf.com/blog/citedrive-easy-reference-management-for-overleaf) und [RStudio](https://citedrive.medium.com/bibliography-management-in-r-markdown-with-citedrive-and-rstudio-2585699dd619) synchronisiert. Sie bieten eine [Browsererweiterung](https://citedrive.medium.com/the-citedrive-companion-for-overleaf-and-rstudio-d8022adb9370) an, die es Ihnen ermöglicht, wissenschaftliche Artikel direkt im Browser und im korrekten BibTeX-Format zu speichern, so dass Sie sie z. B. direkt im Overleaf-Projekt zitieren können, ohne zwischen Browserfenstern/Tabs wechseln zu müssen. Die Erweiterung kann auch Webseiten im BibLaTeX-Format speichern, wobei automatisch der Titel, die URL und das Zugriffsdatum erfasst werden.
:::

## Verwendung von BibTeX

In BibTeX verwenden wir `@misc` als Workaround und nutzen das Feld `note`, um die URl und normalerweise ein Zugriffsdatum anzugeben. In diesem Beispiel wird `url` (*oder `hyperref`*) mit `usepackage{url}` benötigt.

```latex
@misc{CiteDrive2022,
	title        = {CiteDrive brings reference management to Overleaf},
	author       = {CiteDrive, Inc},
	year         = 2022,
	note         = {\url{https://www.citedrive.com/overleaf} [Accessed: (Use the date of access)]}
}
```


Alternativ kann auch das Feld "howpublished" verwendet werden:

```latex
@misc{CiteDrive2022,
	title        = {CiteDrive brings reference management to Overleaf},
	author       = {CiteDrive, Inc},
	year         = 2022,
	note         = {Accessed: (Use the date of access)},
	howpublished = {\url{https://www.citedrive.com/overleaf}}
}
```


## Verwendung von BibLaTeX

Wenn Sie BibLaTeX verwenden, was wir empfehlen, wird der Eintragstyp `@online` von Haus aus unterstützt und funktioniert ohne weitere Anpassungen. Sie können die URL und das Zugriffsdatum in die Felder `url` bzw. `urldate` eingeben.

```latex
@online{CiteDrive2022,
	title        = {CiteDrive brings reference management to Overleaf},
	author       = {CiteDrive, Inc},
	year         = 2022,
	url          = {https://www.citedrive.com/overleaf},
	urldate      = {Use the date of access}
}
```

## Verwendung von `natbib`

Häufig wird nachgefragt, wie man Webseiten mit [natbib](/natbib) zitieren kann. Wir betonen, dass es auf die gleiche Weise wie BibTeX funktioniert. Natbib ist ein Werkzeug zur Gestaltung von In-Text-Zitaten, das auf BibTeX als Backend aufbaut.
