Das Feld `series` in BibTeX wird verwendet, um die Serie anzugeben, zu der ein Buch, Bericht oder eine andere Publikation gehört. Die Angabe der Serie kann zusätzlichen Kontext für den Leser bieten, da sie möglicherweise mit einem breiteren thematischen oder wissenschaftlichen Diskurs in Verbindung steht.

<CiteDrive />

Dieses Feld ist optional und kann in verschiedenen Eintragstypen verwendet werden, einschließlich, aber nicht beschränkt auf `@book`, `@inbook`, `@incollection` und `@inproceedings`. Es ist besonders nützlich, wenn das zitierte Werk Teil einer fortlaufenden Serie oder eines größeren Werkes ist, das ein gemeinsames Thema, einen gemeinsamen Autor oder Herausgeber teilt.

Bei der Eingabe von Daten in das Feld `series` wird empfohlen, den vollständigen offiziellen Titel der Serie zu verwenden. Wenn die Serie eine ISSN hat, kann es auch relevant sein, diese für eine genauere Referenzierung anzugeben.

## Format

Das allgemeine Format für die Angabe einer Serie in Ihrem BibTeX-Eintrag lautet wie folgt:

```bibtex
@entrytype{key,
  ...
  series = {Name der Serie},
  ...
}
```

## Beispiel

Hier ist ein Beispiel, wie das Feld `series` in einem `@book`-Eintrag verwendet wird:

```bibtex
@book{doe2023,
  author = {John Doe},
  title = {Fortgeschrittene Studien in Quantenmechanik},
  year = {2023},
  publisher = {Scientific Publishing Ltd.},
  series = {Cambridge Advanced Series in Physics},
}
```

In diesem Beispiel ist das Buch Teil der "Cambridge Advanced Series in Physics," die verschiedene Titel zu fortgeschrittenen Themen im Bereich der Physik umfassen könnte. Die Angabe der Serie hilft, das Werk innerhalb eines größeren wissenschaftlichen Diskurses einzuordnen und kann den Lesern zusätzliche Ressourcen zum Thema bieten.
