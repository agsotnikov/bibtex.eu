In der akademischen und Verlagswelt bezeichnet die Ausgabe eines Buches die Version oder Form, in der ein Werk veröffentlicht wurde. Das `edition`-Feld in einem BibTeX-Eintrag wird verwendet, um dieses Detail anzugeben, häufig um die Überarbeitungen oder Aktualisierungen hervorzuheben, die das Werk seit seiner ursprünglichen Veröffentlichung durchlaufen hat. Die Angabe der Ausgabe ist besonders wichtig, wenn Sie Lehrbücher oder wissenschaftliche Werke zitieren, die in mehreren Auflagen erschienen sind. Im Folgenden erläutern wir, wie Sie das `edition`-Feld in Ihren BibTeX-Einträgen effektiv nutzen.

### Überblick

Das `edition`-Feld wird verwendet, um die Ausgabe eines Buches anzugeben. Diese Information ist wesentlich, wenn das Buch, das Sie zitieren, in verschiedenen Ausgaben veröffentlicht wurde, die möglicherweise unterschiedliche Inhalte oder Seitenzahlen haben. Die Angabe der Ausgabe hilft dabei, eine klare und genaue Referenz bereitzustellen, und unterstützt die Leser bei der Auffindung der genauen Version des Werks, das Sie verwendet haben.

### Richtlinien

1. **Formatierung**: Es wird empfohlen, die Ausgabe als Ordinalzahl (z.B. „Erste“, „Zweite“, „Dritte“ usw.) mit großgeschriebenem Anfangsbuchstaben einzugeben.
2. **Ausschreibung**: Statt numerischer Darstellungen (1st, 2nd, 3rd) wird das Ausschreiben der Ordinalzahlen für bessere Lesbarkeit bevorzugt.
3. **Sprachliche Anpassung**: Für Werke in anderen Sprachen kann die Ausgabe in dieser Sprache angegeben werden (z.B. „Erste“ für die erste Auflage auf Deutsch).

### Syntax und Struktur

Die Syntax für das `edition`-Feld ist einfach, wie im Folgenden dargestellt:

```bibtex
@book{citationkey,
  ...
  edition={OrdinalNumber},
  ...
}
```

### Beispiel

Hier ist ein Beispiel für einen BibTeX-Eintrag, der das `edition`-Feld enthält:

```bibtex
@book{author2023title,
  title={Der Titel des Buches},
  author={Autor, A.},
  year={2023},
  publisher={Verlagshaus},
  edition={Erste},
}
```

In diesem Beispiel gibt das `edition`-Feld klar an, dass das zitierte Werk die erste Auflage des Buches ist.

### Zusätzliche Tipps

1. **Überarbeitete Ausgaben**: Bei Büchern mit überarbeiteten Ausgaben können Sie auch Begriffe wie „Überarbeitet“ oder „Aktualisiert“ zusammen mit der Ausgabennummer verwenden (z.B. „Zweite, Überarbeitete“).
2. **Spätere Ausgaben**: Wenn Sie aus einer viel späteren Ausgabe zitieren, kann es angebracht sein, in Ihrer Zitation eine Notiz einzufügen, die die Wahl erklärt, wenn die Ausgabe erheblich unterschiedliche oder aktualisierte Inhalte enthält.

### Fazit

Die korrekte Angabe des `edition`-Feldes in Ihren BibTeX-Einträgen sorgt für einen genaueren und zuverlässigeren Zitierprozess. Sie hilft dabei, Ihre Leser auf die exakte Version des Quellmaterials zu lenken, das Sie verwendet haben, und fördert so Klarheit und wissenschaftliche Integrität. Nutzen Sie dieses Feld weise, um die Qualität und Zuverlässigkeit Ihrer bibliografischen Referenzen zu verbessern.
