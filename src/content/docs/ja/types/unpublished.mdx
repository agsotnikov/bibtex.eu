BibTeXは、研究者や学者が参考文献を整理しフォーマットするのを助ける人気のある文献管理システムです。コンピュータサイエンスで広く使用されていますが、他の多くの分野でも文献の引用が必要な場合に役立ちます。BibTeXの「unpublished」エントリタイプは、論文や学位論文、技術報告書など、正式に出版されていない資料を引用するために使用されます。

「unpublished」エントリタイプは、著者名、タイトル、機関、日付など、未発表の資料に関するすべての関連情報を含めることができる柔軟なエントリタイプです。未発表の資料に対するBibTeXエントリの例は以下の通りです：

<CiteDrive/>

```bibtex
@unpublished{key,   
	author = {ジョン・ドウ},   
	title = {未発表資料の重要性},   
	institution = {例大学},   
	year = {2022}, 
}
```

この例での「key」フィールドは、文書内で資料を参照するための一意の識別子です。「author」フィールドには資料の著者名が、「title」フィールドには資料のタイトルが、「institution」フィールドには資料が作成された機関の名前が、「year」フィールドには資料が作成された年が含まれています。

「unpublished」エントリタイプは、正式に出版されていない資料にのみ使用するべきです。たとえオンラインでのみ公開されていても、出版された資料は「techreport」「mastersthesis」または「phdthesis」として引用する必要があります。
