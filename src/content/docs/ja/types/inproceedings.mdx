BibTeXは、LaTeX文書で一般的に使用される参考文献管理ツールです。`inproceedings` BibTeXタイプは、学会の論文集に発表された論文に使用されます。このガイドでは、`inproceedings` BibTeXタイプの必須フィールドとオプションフィールドについて説明します。

<CiteDrive/>

## 必須フィールド

`inproceedings` BibTeXタイプには、次のフィールドが必要です：

-   `author`: 論文の著者。
-   `title`: 論文のタイトル。
-   `booktitle`: 学会の論文集のタイトル。
-   `year`: 学会の論文集が出版された年。

## オプションフィールド

必須フィールドに加えて、`inproceedings` BibTeXタイプには追加の情報を提供するためのいくつかのオプションフィールドがあります。これらのフィールドには以下が含まれます：

-   `editor`: 学会の論文集の編集者。
-   `volume`: 学会の論文集の巻数（該当する場合）。
-   `number`: 学会の論文集の号数（該当する場合）。
-   `series`: 学会の論文集が属するシリーズ名（該当する場合）。
-   `pages`: 論文が学会の論文集に掲載されているページ。
-   `address`: 学会が開催された場所。
-   `month`: 学会が開催された月。
-   `organization`: 学会を主催した組織。
-   `publisher`: 学会の論文集の出版社。

## 例

以下は、`inproceedings` BibTeXタイプの使用例です：

```bibtex
@inproceedings{mypaper,
	title        = {例の分析},
	author       = {ジョン・スミス と ジェーン・ドウ},
	year         = 2022,
	month        = {6月},
	booktitle    = {国際会議「例」論文集},
	publisher    = {ACM Press},
	address      = {ニューヨーク, NY},
	series       = {例会議論文集},
	volume       = 1,
	number       = 2,
	pages        = {123--130},
	editor       = {デイビッド・ブラウン と スーザン・グリーン},
	organization = {ACM}
}
```

この例では、BibTeXエントリは「例の分析」と題されたジョン・スミスとジェーン・ドウによる論文を定義しています。この論文は国際会議「例」の論文集に掲載されており、2022年に出版されました。編集者はデイビッド・ブラウンとスーザン・グリーンで、学会は6月にニューヨークで開催され、ACMがスポンサーです。論文は「例会議論文集」シリーズの第1巻、第2号の123ページから130ページに掲載されています。
