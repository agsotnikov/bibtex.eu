BibTeXは、LaTeX文書で一般的に使用される参考文献管理ツールです。`phdthesis` BibTeXタイプは、博士論文や学位論文に使用されます。このガイドでは、`phdthesis` BibTeXタイプの必須フィールドとオプションフィールドについて説明します。

<CiteDrive/>

## 必須フィールド

`phdthesis` BibTeXタイプには、次のフィールドが必要です：

-   `author`: 論文の著者。
-   `title`: 論文のタイトル。
-   `school`: 学位を授与した機関の名前。
-   `year`: 学位が授与された年。

## オプションフィールド

必須フィールドに加えて、`phdthesis` BibTeXタイプには追加の情報を提供するためのいくつかのオプションフィールドがあります。これらのフィールドには以下が含まれます：

-   `type`: 論文の種類（例：「PhD thesis」または「Master's thesis」）。
-   `address`: 機関の所在地。
-   `month`: 論文が提出された月。
-   `note`: 論文に関する追加情報。

## 例

以下は、`phdthesis` BibTeXタイプの使用例です：

```bibtex
@phdthesis{mythesis,
	title        = {例の分析},
	author       = {ジョン・スミス},
	year         = 2022,
	month        = {6月},
	address      = {例市, CA},
	note         = {こちらからアクセス可能：\url{https://example.com/thesis.pdf}},
	school       = {例大学},
	type         = {PhD thesis}
}
```

この例では、BibTeXエントリは「例の分析」と題されたジョン・スミスによる博士論文を定義しています。学位は2022年に例大学から授与され、論文は6月に例市、CAで提出されました。論文の種類は「PhD thesis」と指定され、論文へのURLが含まれたノートも追加されています。
