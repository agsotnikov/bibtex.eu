BibTeXにはさまざまなエントリタイプがありますが、時には事前定義されたタイプのいずれにも完全に一致しない資料に出会うことがあります。このような場合には、`misc`タイプを使用します。このタイプは、他のカテゴリに該当しないすべての資料に適しています。音楽作品や講義スライドなどが含まれることもありますが、主にウェブページを引用するために使用されます。これらのサポートされていないソースタイプは、`note`や`howpublished`からのワークアラウンドフィールドを使用して作成されます。BibTeXがサポートしていないソースタイプを頻繁に引用する場合は、フィールドやタイプに関してより多くの選択肢を提供するBibLaTeXを検討することをお勧めします。

<CiteDrive/>

```latex

@misc{citationKey,

	...

}

```

ウェブサイトを引用するために`misc`を使用する方法については、このガイドを参照してください。
