BibTeXは、LaTeX文書で一般的に使用される参考文献管理ツールです。`proceedings` BibTeXタイプは、会議で発表された論文集など、会議の議事録に使用されます。このガイドでは、`proceedings` BibTeXタイプの必須フィールドとオプションフィールドについて説明します。

<CiteDrive/>

## 必須フィールド

`proceedings` BibTeXタイプには、次のフィールドが必要です：

-   `title`: 会議の議事録のタイトル。
-   `year`: 会議の議事録が出版された年。

## オプションフィールド

必須フィールドに加えて、`proceedings` BibTeXタイプには追加の情報を提供するためのいくつかのオプションフィールドがあります。これらのフィールドには以下が含まれます：

-   `editor`: 会議の議事録の編集者の名前。
-   `volume`: 会議の議事録の巻号（該当する場合）。
-   `number`: 会議の議事録の号数（該当する場合）。
-   `series`: 会議の議事録が属するシリーズ名（該当する場合）。
-   `address`: 会議の開催場所。
-   `month`: 会議が開催された月。
-   `publisher`: 会議の議事録の出版社名。

## 例

以下は、`proceedings` BibTeXタイプの使用例です：

```bibtex
@proceedings{myproceedings,
	title        = {国際会議「Example」の議事録},
	year         = 2022,
	month        = {6月},
	publisher    = {ACM Press},
	address      = {ニューヨーク, NY},
	series       = {Example Conference Proceedings},
	volume       = 1,
	number       = 2,
	editor       = {ジェーン・ドウ と ジョン・スミス}
}
```

この例では、BibTeXエントリは「国際会議「Example」の議事録」と題された会議の議事録セットを定義しています。議事録は2022年に出版され、ジェーン・ドウとジョン・スミスによって編集されました。会議は6月にニューヨークで開催され、議事録はACM Pressから「Example Conference Proceedings」シリーズの第1巻第2号として出版されました。
