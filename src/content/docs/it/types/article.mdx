BibTeX è un sistema di gestione delle referenze utilizzato per formattare le liste di referenze in un documento. È comunemente usato nella scrittura accademica, in particolare nei campi scientifici e tecnici. In questa guida, introdurremo il tipo di voce `article` in BibTeX e ti mostreremo come usarlo nei tuoi documenti LaTeX.

<CiteDrive/>

## Il Tipo di Voce Article

Il tipo di voce `article` è utilizzato per riferirsi agli articoli pubblicati in riviste accademiche, magazine o giornali. I campi richiesti per una voce `article` sono:

-   `author`: Il nome(nomi) dell'autore(o) dell'articolo.
-   `title`: Il titolo dell'articolo.
-   `journal`: Il nome della rivista, del magazine o del giornale in cui è stato pubblicato l'articolo.
-   `year`: L'anno di pubblicazione.

I campi opzionali per una voce `article` includono:

-   `volume`: Il numero del volume della rivista.
-   `number`: Il numero dell'edizione della rivista.
-   `pages`: I numeri di pagina su cui appare l'articolo.
-   `month`: Il mese di pubblicazione.
-   `note`: Informazioni varie.

Ecco un esempio di una voce `article`:

```bibtex
@article{example_article,
	title        = {Gli Effetti del Cambiamento Climatico},
	author       = {John Smith},
	year         = 2022,
	month        = {Gennaio},
	journal      = {Scientific American},
	volume       = 327,
	number       = 1,
	pages        = {44--49},
	note         = {Questo è un esempio di voce per un articolo in una rivista.}
}
```

## Utilizzare il Tipo di Voce Article in LaTeX

Una volta creata una voce `article` nel tuo file BibTeX, puoi riferirti ad essa nel tuo documento LaTeX utilizzando il comando `\cite`. Ad esempio:

```latex
\documentclass{article}  
\begin{document}  
Secondo \cite{example_article}, il cambiamento climatico ha effetti significativi sul pianeta.  
\bibliography{references}
\bibliographystyle{plain} 
\end{document}
```

In questo esempio, la voce `example_article` nel file `references.bib` è citata utilizzando il comando `\cite`. La bibliografia viene poi generata utilizzando lo stile `plain`.
