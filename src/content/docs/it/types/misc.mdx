Esistono vari tipi di voce in BibTeX e a volte puoi imbatterti in una fonte che non si adatta perfettamente a nessuno dei tipi predefiniti. In questi casi, puoi utilizzare il tipo `misc`. Questo tipo è per tutto ciò che non rientra in una delle altre categorie. Questo potrebbe includere pezzi musicali o diapositive di conferenze, ma principalmente è utilizzato per citare pagine web. Questi tipi di fonte non supportati sono creati utilizzando campi di lavoro come `note` e `howpublished`. Se citi frequentemente tipi di fonte non supportati da BibTeX, dovresti considerare di passare a BibLaTeX, che offre un insieme più ampio di campi e tipi.

<CiteDrive/>

```latex

@misc{citationKey,

	...

}

```

Se desideri utilizzare `misc` per citare un sito web, consulta questa guida.
