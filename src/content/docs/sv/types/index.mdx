BibTeX är ett populärt verktyg för hantering av bibliografiska referenser som används inom datavetenskap och andra tekniska discipliner. Det används för att lagra och hantera bibliografiska referenser, såsom böcker, artiklar och konferenspapper, i ett strukturerat format. Detta format kan sedan användas för att automatiskt generera bibliografier i olika dokumentformat.

<CiteDrive/>
