BibTeX 中有多种条目类型，有时你会遇到不完全符合任何预定义类型的来源。在这些情况下，你可以使用 `misc` 类型。此类型适用于不属于其他类别的任何内容。它可以用于引用音乐作品或讲义幻灯片，但通常用于引用网页。这些不支持的来源类型通过使用 `note` 和 `howpublished` 字段来创建。如果你经常引用 BibTeX 不支持的来源类型，你可以考虑使用 BibLaTeX，它在字段和类型上提供了一个更大的超集。

<CiteDrive/>

```latex

@misc{citationKey,

	...

}

```

如果你打算使用 `misc` 来引用网站，请查看本指南。
