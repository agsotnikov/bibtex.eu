BibTeX 是一种常用于 LaTeX 文档中的参考管理工具。"phdthesis" 是用于引用博士论文的 BibTeX 类型。在本指南中，我们将解释 "phdthesis" BibTeX 类型的必填字段和可选字段。

<CiteDrive/>

## 必填字段

"phdthesis" BibTeX 类型要求填写以下字段：

-   `author`: 论文的作者。
-   `title`: 论文的标题。
-   `school`: 授予学位的机构名称。
-   `year`: 学位授予的年份。

## 可选字段

除了必填字段外，"phdthesis" BibTeX 类型还包含许多可选字段，用于提供更多信息。这些字段包括：

-   `type`: 论文的类型，如 "PhD thesis" 或 "Master's thesis"。
-   `address`: 机构的所在地。
-   `month`: 论文提交的月份。
-   `note`: 关于论文的其他信息。

## 示例

以下是使用 "phdthesis" BibTeX 类型的示例：

```bibtex
@phdthesis{mythesis,
	title        = {An Analysis of Example},
	author       = {John Smith},
	year         = 2022,
	month        = {June},
	address      = {Example City, CA},
	note         = {Available at \url{https://example.com/thesis.pdf}},
	school       = {University of Example},
	type         = {PhD thesis}
}
```

在此示例中，BibTeX 条目定义了一篇由 John Smith 撰写的题为 "An Analysis of Example" 的博士论文。学位于 2022 年由 Example 大学授予，论文于六月在 Example City, CA 提交。论文的类型被指定为 "PhD thesis"，并附有提供论文链接的备注。
