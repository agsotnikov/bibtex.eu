BibTeX — это система управления ссылками для форматирования списков ссылок в документе. Она широко используется в академическом сообществе, особенно в областях науки, технологии и математики.

Один из типов записей, которые можно включить в файл BibTeX, — это тип записи `booklet`. В этом руководстве мы познакомим вас с типом записи `booklet` и покажем, как использовать его для управления ссылками в ваших документах LaTeX.

## Тип записи Booklet

Тип записи `booklet` используется для ссылок на печатные работы, которые не были формально опубликованы, но скреплены вместе, как книга. Примеры записей `booklet` могут включать материалы конференций, программные руководства или инструкции.

Обязательные поля для записи `booklet` следующие:

-   `title`: Заголовок буклета.
-   `author`: Автор или авторы буклета.
-   `howpublished`: Как был опубликован буклет.
-   `address`: Адрес издателя или спонсирующей организации.
-   `year`: Год публикации буклета.

Дополнительные поля для записи `booklet` включают:

-   `editor`: Редактор или редакторы буклета.
-   `volume`: Номер тома многотомной книги.
-   `number`: Номер журнала или серии.
-   `series`: Название серии или набора книг.
-   `organization`: Организация, которая издала или спонсировала буклет.
-   `month`: Месяц публикации.
-   `note`: Разная информация.

Вот пример записи `booklet`:

```bibtex
@booklet{example_booklet,
	title        = {Материалы конференции},
	author       = {Джейн Доу и Джон Смит},
	year         = 2022,
	month        = {Июль},
	address      = {Сан-Франциско, CA},
	note         = {Это пример записи для буклета материалов конференции.},
	howpublished = {Представлено на Ежегодной конференции по технологиям},
	editor       = {Боб Джонсон}
}
```

## Использование типа записи Booklet в LaTeX

После того как вы создали запись `booklet` в вашем файле BibTeX, вы можете сослаться на нее в документе LaTeX, используя команду `\cite`. Например:

```latex
\documentclass{article}  
\begin{document}  
Согласно \cite{example_booklet}, Ежегодная конференция по технологиям прошла успешно.  
\bibliography{references} 
\bibliographystyle{plain}  
\end{document}
```

В этом примере запись `example_booklet` в файле `references.bib` цитируется с помощью команды `\cite`. Библиография затем создается с использованием стиля `plain`.
