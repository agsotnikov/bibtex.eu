BibTeX — это система управления ссылками, используемая для форматирования списков ссылок в документе. Она часто используется в академическом письме, особенно в научных и технических областях. В этом руководстве мы познакомим вас с типом записи `article` в BibTeX и покажем, как использовать его в ваших документах LaTeX.

## Тип записи Article

Тип записи `article` используется для ссылки на статьи, опубликованные в научных журналах, журналах или газетах. Обязательные поля для записи типа `article` включают:

-   `author`: Имя(а) автора(ов) статьи.
-   `title`: Заголовок статьи.
-   `journal`: Название журнала, в котором была опубликована статья.
-   `year`: Год публикации.

Дополнительные поля для записи типа `article` включают:

-   `volume`: Номер тома журнала.
-   `number`: Номер выпуска журнала.
-   `pages`: Номера страниц, на которых находится статья.
-   `month`: Месяц публикации.
-   `note`: Разная информация.

Вот пример записи типа `article`:

```bibtex
@article{example_article,
	title        = {Влияние изменения климата},
	author       = {Джон Смит},
	year         = 2022,
	month        = {Январь},
	journal      = {Scientific American},
	volume       = 327,
	number       = 1,
	pages        = {44--49},
	note         = {Это пример записи для статьи в журнале.}
}
```

## Использование типа записи Article в LaTeX

После того как вы создали запись типа `article` в вашем файле BibTeX, вы можете сослаться на нее в вашем документе LaTeX, используя команду `\cite`. Например:

```latex
\documentclass{article}  
\begin{document}  
Согласно \cite{example_article}, изменение климата оказывает значительное влияние на планету.  
\bibliography{references}
\bibliographystyle{plain} 
\end{document}
```

В этом примере запись `example_article` в файле `references.bib` цитируется с помощью команды `\cite`. Затем библиография создается с использованием стиля `plain`.
