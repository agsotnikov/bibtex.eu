BibTeX — это инструмент для управления ссылками, который часто используется в документах LaTeX. Тип записи BibTeX "masterthesis" используется для магистерских диссертаций. В этом руководстве мы объясним обязательные и дополнительные поля для типа записи "masterthesis" в BibTeX.

<CiteDrive/>

## Обязательные поля

Тип записи "masterthesis" в BibTeX требует следующих полей:

-   `author`: Автор диссертации.
-   `title`: Название диссертации.
-   `school`: Название учебного заведения, присудившего степень.
-   `year`: Год присуждения степени.

## Дополнительные поля

В дополнение к обязательным полям, тип записи "masterthesis" в BibTeX также имеет несколько дополнительных полей, которые можно использовать для предоставления дополнительной информации. Эти поля включают:

-   `type`: Тип диссертации, например, "Магистерская диссертация".
-   `address`: Местоположение учебного заведения.
-   `month`: Месяц подачи диссертации.
-   `note`: Любая дополнительная информация о диссертации.

## Пример

Вот пример того, как использовать тип записи "masterthesis" в BibTeX:

```bibtex
@masterthesis{mythesis,
	title        = {A Study of Example},
	author       = {Jane Doe},
	year         = 2022,
	month        = {June},
	address      = {Example City, CA},
	note         = {Available at \url{https://example.com/thesis.pdf}},
	school       = {University of Example},
	type         = {Master's thesis}
}
```

В этом примере запись BibTeX определяет магистерскую диссертацию, написанную Джейн Доу, под названием "A Study of Example". Степень была присуждена в 2022 году Университетом Example, а диссертация была подана в июне в городе Example, штат Калифорния. Тип диссертации указан как "Магистерская диссертация", а в примечании добавлена ссылка на документ.
