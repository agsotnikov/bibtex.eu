BibTeX — это инструмент для управления ссылками, который часто используется в документах LaTeX. Тип записи BibTeX "phdthesis" используется для цитирования докторских диссертаций или кандидатских работ. В этом руководстве мы объясним обязательные и дополнительные поля для типа записи "phdthesis" в BibTeX.

## Обязательные поля

Тип записи "phdthesis" в BibTeX требует следующих полей:

-   `author`: Автор диссертации.
-   `title`: Название диссертации.
-   `school`: Название учебного заведения, присудившего степень.
-   `year`: Год присуждения степени.

## Дополнительные поля

В дополнение к обязательным полям, тип записи "phdthesis" в BibTeX также имеет несколько дополнительных полей, которые можно использовать для предоставления дополнительной информации. Эти поля включают:

-   `type`: Тип диссертации, например, "Докторская диссертация" или "Магистерская диссертация".
-   `address`: Местоположение учебного заведения.
-   `month`: Месяц подачи диссертации.
-   `note`: Любая дополнительная информация о диссертации.

## Пример

Вот пример того, как использовать тип записи "phdthesis" в BibTeX:

```bibtex
@phdthesis{mythesis,
	title        = {An Analysis of Example},
	author       = {John Smith},
	year         = 2022,
	month        = {June},
	address      = {Example City, CA},
	note         = {Available at \url{https://example.com/thesis.pdf}},
	school       = {University of Example},
	type         = {PhD thesis}
}
```

В этом примере запись BibTeX определяет докторскую диссертацию, написанную Джоном Смитом, под названием "An Analysis of Example". Степень была присуждена в 2022 году Университетом Example, а диссертация была подана в июне в городе Example, штат Калифорния. Тип диссертации указан как "Докторская диссертация", а в примечании добавлена ссылка на документ.
