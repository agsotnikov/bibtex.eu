В BibTeX существует множество типов записей, и иногда вам может попасться источник, который не совсем подходит ни под один из предопределённых типов. В таких случаях можно использовать тип `misc`. Этот тип предназначен для всего, что не подходит под другие категории. Это могут быть музыкальные произведения или презентации, но чаще всего он используется для цитирования веб-страниц. Эти неподдерживаемые типы источников создаются с помощью полей `note` и `howpublished`. Если вы часто цитируете типы источников, не поддерживаемые BibTeX, вам стоит обратить внимание на BibLaTeX, который предлагает расширенный набор полей и типов.

<CiteDrive/>

```latex
@misc{citationKey,

	...

}
```

Если вы хотите использовать тип `misc` для цитирования веб-сайта, обратитесь к этому руководству.
