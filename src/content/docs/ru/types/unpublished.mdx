BibTeX — это популярная система управления библиографией, которая помогает исследователям и учёным организовывать и форматировать их библиографии. Она широко используется в информатике, но также полезна и в других областях, где требуется цитирование источников. Тип записи "unpublished" в BibTeX используется для цитирования источников, которые официально не опубликованы, таких как диссертации, дипломные работы и технические отчеты.

Тип записи "unpublished" является универсальным и позволяет авторам включать всю необходимую информацию об неопубликованном источнике, такую как имя автора, название, учреждение и дата. Пример записи BibTeX для неопубликованного источника выглядит следующим образом:

<CiteDrive/>

```bibtex
@unpublished{key,   
	author = {John Doe},   
	title = {The Importance of Unpublished Sources},   
	institution = {University of Example},   
	year = {2022}, 
}
```

Поле "key" в этом примере — это уникальный идентификатор, который используется для ссылки на источник в тексте документа. Поле "author" содержит имя(имена) автора(ов) источника, поле "title" содержит название источника, поле "institution" содержит название учреждения, где был написан источник, а поле "year" содержит год написания источника.

Следует отметить, что тип записи "unpublished" должен использоваться только для источников, которые ещё не были официально опубликованы. Если источник был опубликован, даже если только в интернете, его следует цитировать как "techreport", "mastersthesis" или "phdthesis", в зависимости от типа публикации.
