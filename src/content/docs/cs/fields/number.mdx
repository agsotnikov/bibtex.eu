V BibTeX se pole `number` používá k určení čísla vydání časopisu, čísla technického reportu nebo jiných identifikátorů, které jsou součástí větší série. Toto pole pomáhá přesněji určit vydání nebo edici série, kde lze nalezený dokument najít, a poskytuje tak přesnější informace pro lokalizaci materiálu.

<CiteDrive />

Toto pole se často používá spolu s polem `volume`, zejména pro články v časopisech a periodikách. U technického reportu by pole `number` obvykle obsahovalo číslo reportu přidělené institucí nebo organizací, která report publikovala.

## Formát

Formát pole `number` je relativně jednoduchý. V kontextu záznamu článku by to vypadalo následovně:

```bibtex
@article{citation_key,
  ...
  number={issue_number},
  ...
}
```

Zde "citation_key" představuje jedinečný identifikátor záznamu a "issue_number" představuje číslo vydání časopisu, technického reportu nebo jiného publikace.

## Příklady

Níže jsou uvedeny příklady ukazující použití pole `number` v různých kontextech:

### Příklad 1: Článek v Časopise

```bibtex
@article{doe2023study,
  author = {Doe, Jane},
  title = {Komplexní studie o kvantovém počítání},
  journal = {Časopis pokročilého počítání},
  year = {2023},
  volume = {29},
  number = {3},
  pages = {215-230},
}
```

V tomto příkladu pole `number` specifikuje třetí číslo 29. ročníku "Časopisu pokročilého počítání".

### Příklad 2: Technický Report

```bibtex
@techreport{smith2022analysis,
  author = {Smith, John},
  title = {Analýza trendů v obnovitelných zdrojích energie},
  institution = {Institut pro výzkum energie},
  year = {2022},
  number = {TR-2022-45},
  type = {Technický report},
}
```

V tomto případě pole `number` používáme k určení specifického čísla reportu přiděleného "Institutem pro výzkum energie".

Přesné používání pole `number` může čtenáři usnadnit nalezení zdrojového materiálu a ověření citovaných informací. Je to důležitý detail ve vědeckém psaní a výzkumné dokumentaci.
