Pole `editor` v BibTeXu se používá k specifikaci editorů publikovaného díla. Toto pole je zásadní pro typy referencí jako jsou knihy, sborníky nebo kolekce, kde je role editora odlišná od autora. Správné uvedení editorů zajišťuje správné přisuzování a hladší citace v akademických pracích a článcích. Zde se podíváme na způsoby, jak uvést editory v BibTeXu.

### Pravidla formátování

V poli `editor` jsou akceptovány různé formáty zápisu. Formátování by mělo být konzistentní a přesné, aby se předešlo nesprávnému zobrazení jména editora. Zde jsou běžně používané zápisy:

#### Zápis: `{Firstname Lastname}`

Toto je jednoduchá metoda, kde uvedete jméno editora následované příjmením. Inverzní formát „Lastname Firstname“ se nedoporučuje.

**Příklad**

```bibtex
@book{Example2023,
  ...
  editor={Isaac Newton},
  ...
}
```

#### Zápis: `{Lastname, Firstname}`

Toto je doporučený formát, který jasně odděluje jméno a příjmení, což je zvláště užitečné pro jména s více komponentami.

**Příklad**

```bibtex
@book{Example2023,
  ...
  editor={Newton, Isaac},
  ...
}
```

#### Zápis: `{Lastname, Suffix, Firstname}`

Pro editory s příponami (např. Junior, Sr., III) by mělo být jméno formátováno s příjmením jako první, následovaným příponou a potom jménem. Všechny komponenty by měly být odděleny čárkami.

**Příklad**

```bibtex
@book{Example2023,
  ...
  editor={King, Jr., Martin Luther},
  ...
}
```

### Přidání více editorů

Pro specifikaci více editorů použijte klíčové slovo `AND` mezi jednotlivými údaji o editorech, formátovanými výše uvedeným způsobem. Tento zápis pomáhá jasně a přesně uvádět několik editorů.

**Příklad**

```bibtex
@book{Example2023,
  ...
  editor={Smith, John AND Doe, Jane},
  ...
}
```

### Nejlepší praktiky

1. **Konzistence**: Udržujte konzistenci ve formátu zápisu napříč všemi záznamy ve vašem BibTeX souboru, aby byla zajištěna jednotná reprezentace.
2. **Přípony**: Jasně specifikujte přípony, aby se předešlo záměnám, zejména u prací s více editory, kteří mají rodinné vztahy.

### Závěr

Pole `editor` je kritickou součástí BibTeX záznamů, zejména pro editované svazky, sborníky a kolekce. Přesné a konzistentní formátování zajišťuje, že editoři obdrží správné přisuzování, čímž zvyšuje důvěryhodnost a čitelnost vašich citací. Nezapomeňte dodržovat doporučené formáty pro hladký citací zážitek.
