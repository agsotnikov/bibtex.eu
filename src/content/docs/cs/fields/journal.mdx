Pole `journal` se používá k určení názvu časopisu, ve kterém byl článek publikován. Toto pole je klíčové pro odlišení článků z různých publikací a poskytuje správný kontext pro citované dílo.

<CiteDrive/>

Zde je šablona, která ukazuje, jak můžete použít pole `journal` v záznamu typu `@article`:

```bibtex
@article{ ... ,
 journal = {Název časopisu},
}

```

V uvedeném formátu nahraďte "Název časopisu" skutečným názvem časopisu, ve kterém byl článek publikován. Je důležité zadat tuto informaci přesně, aby citace byly správné.

Například následuje příklad ukazující správné použití pole `journal`:

```bibtex
@article{smith2023effects,
 title = {Účinky XYZ na ABC},
 author = {Smith, John},
 journal = {Journal of ABC Research},
 year = {2023},
}
```
