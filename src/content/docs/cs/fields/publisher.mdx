Pole `publisher` se používá v různých záznamech BibTeX, jako jsou `@book`, `@inbook` nebo `@incollection`, k určení názvu vydavatele, který publikoval odkazovanou práci. Začlenění vydavatele do citace pomáhá poskytovat komplexní a přesný odkaz, což může čtenářům pomoci při vyhledávání původního zdroje.

<CiteDrive />

Při zadávání vydavatele do vašeho BibTeX záznamu se ujistěte, že používáte celý a oficiální název vydavatele. To zajistí, že citace bude profesionální a důvěryhodná. Dále pomáhá předejít zmatku s jinými vydavateli, kteří mohou mít podobná jména.

## Příklady

Zde jsou některé příklady, jak správně uvést vydavatele v různých záznamech BibTeX:

### Kniha

```bibtex
@book{key1,
  title = {The Art of Scientific Research},
  author = {John Doe},
  year = {2023},
  publisher = {Cambridge University Press},
}
```

### Kapitola v knize

```bibtex
@inbook{key2,
  title = {Chapter on Quantum Mechanics},
  author = {Jane Doe},
  year = {2023},
  publisher = {Oxford University Press},
  pages = {45--78},
  editor = {Sarah Smith},
  booktitle = {Modern Physics},
}
```

### Článek v kolekci

```bibtex
@incollection{key3,
  title = {Essay on Environmental Conservation},
  author = {Alex Johnson},
  year = {2023},
  publisher = {Springer},
  booktitle = {Sustainable Development: Theories and Practices},
  editor = {Mark Lee},
}
```

Upozorňujeme, že pole `publisher` je obvykle doprovázeno dalšími důležitými poli, jako jsou `title`, `author`, `year` atd., aby se poskytl úplný odkaz. Je také dobré poznamenat, že v závislosti na používaném citačním stylu se formát a umístění pole vydavatele mohou lišit, takže vždy dodržujte pokyny specifického stylu, který používáte.
