## Vytvoření souboru .BIB ručně

Vytvoření souboru `.BIB` je vlastně velmi jednoduché. Stačí vytvořit textový soubor s příponou `.bib`. Do tohoto souboru můžete přidat své BibTeX reference, jednu na řádek. Například:

```latex
	@article{smith201X,
		title        = {Zajímavý článek},
		author       = {John Smith},
		year         = {201X},
		journal      = {Žurnál zajímavých článků}
	}
```

Jakmile máte svůj `.BIB` soubor vytvořený, můžete ho použít ve svých dokumentech LaTeX pomocí příkazu `\bibliography{filename}`. Například:

```latex
\documentclass{article}
\usepackage{natbib}

\begin{document}

...

\bibliography{filename}

\end{document}
```

To je vše! Vytvoření souboru `.BIB` je jednoduchý způsob, jak spravovat své BibTeX reference. Nezapomeňte se podívat na naše další články o BibTeX pro více tipů a triků!
