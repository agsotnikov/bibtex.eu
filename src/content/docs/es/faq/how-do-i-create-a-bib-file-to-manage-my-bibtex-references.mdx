## Crear un archivo bib a mano

Crear un archivo `.BIB` es en realidad bastante simple. Solo necesitas crear un archivo de texto con la extensión `.bib`. Luego, puedes añadir tus referencias BibTeX a este archivo, una por línea. Por ejemplo:

```latex
	@article{smith201X,
		title        = {Un artículo interesante},
		author       = {John Smith},
		year         = {201X},
		journal      = {Revista de Artículos Interesantes}
	}
```

Una vez que hayas creado tu archivo `.BIB`, puedes usarlo en tus documentos LaTeX especificando el comando `\bibliography{filename}`. Por ejemplo:

```latex
\documentclass{article}
\usepackage{natbib}

\begin{document}

...

\bibliography{filename}

\end{document}
```

¡Y eso es todo! Crear un archivo `.BIB` es una manera sencilla de gestionar tus referencias BibTeX. Asegúrate de consultar nuestros otros artículos sobre BibTeX para más consejos y trucos.
