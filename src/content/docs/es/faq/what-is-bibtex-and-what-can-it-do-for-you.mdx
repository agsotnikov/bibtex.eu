---
title: "¿Qué es BibTeX y qué puede hacer por ti?"
---

BibTeX es una herramienta de gestión de bibliografías para LaTeX que facilita la creación y gestión de bibliografías. Puede crear citas y referencias para artículos, libros, sitios web, etc. Los archivos BibTeX tienen la extensión de archivo .bib.

<CiteDrive />

La herramienta se utiliza principalmente para crear bibliografías y citas para documentos LaTeX, un sistema de tipografía a menudo utilizado por matemáticos, físicos y científicos de la computación, o estudiantes de estas disciplinas. Sin embargo, LaTeX también ha encontrado su camino en otros campos como la psicología, biología, sociología e incluso el derecho (ver jurabib). También se utiliza para sistemas basados en Pandoc, como R Markdown.

Mientras que algunos mantienen los datos bibliográficos a mano, LaTeX se usa principalmente en conjunto con software de gestión de literatura como CiteDrive o JabRef, ambos desarrollados para satisfacer las necesidades de gestión de referencias BibTeX y facilitar la búsqueda en línea, la importación y la colaboración para una mejor gestión de los datos bibliográficos.
