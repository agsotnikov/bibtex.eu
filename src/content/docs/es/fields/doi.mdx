El campo `doi` es una parte integral de la referencia moderna, permitiendo un método fluido y permanente para localizar objetos digitales como artículos de revistas, informes de investigación y conjuntos de datos en un entorno en línea. Aquí, profundizamos en la funcionalidad y la importancia del campo `doi` en BibTeX, así como en la orientación sobre su uso correcto.

### Visión General

Un Identificador de Objeto Digital (DOI) es una cadena alfanumérica única asignada a un objeto digital, facilitando un enlace persistente a su ubicación en Internet. Como un método estandarizado para identificar recursos electrónicos, el DOI permanece estable incluso si la URL del ítem cambia, asegurando la longevidad de las referencias.

El campo `doi` en BibTeX sirve para registrar este identificador, permitiendo citas precisas y permanentes. Aunque se encuentra predominantemente en el tipo de entrada `article`, también se puede usar en otros tipos como `book`, `chapter`, `inproceedings` o `conference`, aunque con menor frecuencia. Independientemente del tipo de entrada, el campo `doi` es opcional.

### Sintaxis y Estructura

Para incorporar correctamente un DOI en tu entrada BibTeX, sigue el siguiente formato:

```bibtex
@entrytype{clavedecita,
  ...
  doi={https://doi.org/xxxxx},
  ...
}
```
El campo `doi` encapsula la URL completa del DOI, que típicamente comienza con "[https://doi.org/](https://doi.org/)" seguido de una cadena de identificador única.

### Ejemplo

Aquí tienes un ejemplo de una entrada BibTeX que incluye un DOI:

```bibtex
@article{steinbeck2003chemistry,
  title={The Chemistry Development Kit (CDK): An open-source Java library for chemo-and bioinformatics},
  author={Steinbeck, Christoph and Han, Yongquan and Kuhn, Stefan and Horlacher, Oliver and Luttmann, Edgar and Willighagen, Egon},
  journal={Journal of chemical information and computer sciences},
  volume={43},
  number={2},
  pages={493--500},
  year={2003},
  publisher={ACS Publications},
  doi={https://doi.org/10.1021/ci025584y},
}
```

En este ejemplo, el campo `doi` contiene la URL permanente que lleva directamente al artículo citado.

### Consejos de Utilización

1. **Encontrar DOIs**: Para artículos académicos, los DOIs a menudo se pueden encontrar en la primera página del artículo o en la base de datos en línea donde se encuentra el artículo.
2. **Referencia Cruzada**: Utiliza el campo `doi` para referencias cruzadas en tu documento, ofreciendo a los lectores una ruta directa al material fuente original.
3. **Opcional pero Beneficioso**: Aunque el campo `doi` es opcional, incluirlo mejora la fiabilidad y rastreabilidad de tus referencias.

### Conclusión

La inclusión del campo `doi` en tus entradas BibTeX representa una buena práctica en la escritura académica, ofreciendo un enlace firme a los objetos digitales que referencias. Al adoptar el sistema DOI, fomentas la accesibilidad y permanencia del conocimiento científico, apoyando a la comunidad investigadora global en sus esfuerzos académicos.

A medida que construyas tu bibliografía, recuerda aprovechar las capacidades del campo `doi`, fomentando un discurso académico más rico, confiable e interconectado.
