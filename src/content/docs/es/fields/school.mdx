En BibTeX, el campo `school` se utiliza predominantemente en las entradas `@phdthesis` o `@mastersthesis`. Este campo se usa para indicar la institución educativa donde se escribió la tesis o disertación. Es un campo obligatorio para ambos tipos de entrada, ayudando a proporcionar información detallada sobre el origen del trabajo académico, lo que puede ser importante para los lectores que deseen buscar más información o referenciar el trabajo en su propia investigación.

## Directrices de Formato

Al ingresar el nombre de la escuela, es aconsejable usar el nombre completo y oficial de la institución para mantener un formato formal y estandarizado en tus referencias. Evita usar abreviaturas a menos que sean ampliamente reconocidas y aceptadas.

## Ejemplos

Aquí tienes ejemplos de cómo usar el campo `school` para las entradas `phdthesis` y `mastersthesis`:

### Tesis de Doctorado

```bibtex
@phdthesis{doe2023,
  author = {John Doe},
  title = {Exploraciones en Computación Cuántica},
  year = {2023},
  school = {Massachusetts Institute of Technology},
}
```

### Tesis de Máster

```bibtex
@mastersthesis{smith2023,
  author = {Sarah Smith},
  title = {Un Estudio Integral sobre la Sostenibilidad Ambiental},
  year = {2023},
  school = {University of California, Berkeley},
}
```

## Nota

En tu ejemplo, usaste el campo `series` en lugar del campo `school` para la entrada `mastersthesis`. Sin embargo, el campo `school` debe utilizarse para especificar la institución en las entradas `phdthesis` y `mastersthesis`, como se muestra en los ejemplos anteriores.
