El campo `pages` en una entrada de BibTeX denota las páginas específicas a las que se hace referencia dentro de un trabajo más grande, como un libro o una revista. Este campo puede ser esencial al citar un capítulo de un libro o un artículo en una revista, ayudando a dirigir a los lectores directamente a la sección pertinente del trabajo.

## Directrices de Formato

Puedes especificar una sola página, un rango de páginas o incluso páginas no consecutivas. La forma recomendada de separar los números de página es con comas o doble guiones. Aquí tienes algunas directrices para formatear el campo `pages` correctamente:

1. **Página Única:** Si te refieres a una sola página, simplemente escribe el número de la página.
    
2. **Rango de Páginas:** Para un rango de páginas, debes usar doble guiones (--). Esto indica un rango continuo desde el primer número hasta el segundo.
    
3. **Páginas No Consecutivas:** Si te refieres a páginas no consecutivas, separa los diferentes números de página o rangos con una coma.
    

## Ejemplos

Aquí tienes algunos ejemplos que demuestran cómo usar el campo `pages`:

```bibtex
@book{clave,
  title = {Título del Libro},
  author = {Nombre del Autor},
  year = {2023},
  pages = {33--44},   % Rango de páginas de 33 a 44
}
```

```bibtex
@book{clave,
  title = {Título del Libro},
  author = {Nombre del Autor},
  year = {2023},
  pages = {22, 33--44},   % Página 22 y un rango de 33 a 44
}
```

```bibtex
@book{clave,
  title = {Título del Libro},
  author = {Nombre del Autor},
  year = {2023},
  pages = {17},    % Página única
}
```

## Nota

Es importante tener en cuenta que el campo `pages` no está limitado a las entradas `@book`, y también puede ser utilizado en otros tipos de entrada como `@article` o `@inbook` para denotar las páginas relevantes en esos contextos. Utilizar el campo `pages` correctamente ayuda a crear citas más precisas y útiles.
