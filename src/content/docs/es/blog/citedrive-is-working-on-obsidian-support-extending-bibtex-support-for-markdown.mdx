[CiteDrive](https://www.citedrive.com/), una herramienta para gestionar y recopilar entradas y citas de BibTeX, está desarrollando una extensión para Obsidian, una aplicación de toma de notas con un enfoque único. Permite a los usuarios vincular y conectar rápidamente sus ideas, aumentando la productividad al escribir artículos, y reconocemos la importancia de los investigadores durante la fase de autoría. CiteDrive también ampliará el soporte para Markdown, como ya lo hace con Jupyter Notebook, Quarto y R Markdown, para ayudar a los investigadores a organizar y acceder a sus referencias en aplicaciones como Obsidian.

# ¿Qué es exactamente Obsidian?
Obsidian es una aplicación multiplataforma que te permite organizar y conectar rápidamente tus pensamientos e ideas, todo en markdown. Viene con un conjunto completo de herramientas para crear notas, diagramas, gráficos, documentos y otros tipos de contenido. La aplicación conecta archivos markdown relacionados de manera novedosa, lo que facilita conectar ideas, explorar nuevos caminos y expandir rápidamente los temas.

Es una excelente herramienta para crear mapas de conocimiento desde cero o a partir de notas tomadas previamente.

Obsidian fue diseñado para ser una base de conocimientos personal, pero funciona bien en proyectos colaborativos. Con el soporte de CiteDrive para Obsidian, obtendrás lo mejor de ambos mundos: una potente herramienta de investigación y una eficiente base de conocimientos. ¡Estén atentos para más información!

# ¿Qué es CiteDrive y para qué se utiliza?
CiteDrive es una herramienta de investigación innovadora basada en la nube que revoluciona el almacenamiento y la organización de datos bibliográficos. Esta aplicación te permite almacenar y acceder de manera segura a tus entradas de BibTeX desde cualquier lugar, en cualquier momento, manteniéndose sincronizada con herramientas como Overleaf y RStudio. Organiza las referencias de tu archivo master.bib global y tus equipos en proyectos para facilitar la colaboración con otros investigadores, estudiantes y profesores, y para que todos estén en la misma página. ¡Te invitamos cordialmente a aprovechar los servicios de primera clase de CiteDrive de inmediato!
