BibTeX es una herramienta de gestión de referencias que se utiliza habitualmente en documentos LaTeX. El tipo BibTeX "phdthesis" se utiliza para tesis doctorales. En esta guía explicaremos los campos obligatorios y opcionales del tipo BibTeX "phdthesis".

<CiteDrive/>

## Campos Requeridos

El tipo BibTeX "phdthesis" requiere los siguientes campos:

-   `author`: El autor de la tesis.
-   `title`: El título de la tesis.
-   `school`: El nombre de la institución que otorgó el grado.
-   `year`: El año en que se otorgó el grado.

## Campos Opcionales

Además de los campos requeridos, el tipo BibTeX "phdthesis" también tiene varios campos opcionales que se pueden utilizar para proporcionar información adicional. Estos campos incluyen:

-   `type`: El tipo de la tesis, como "Tesis de doctorado" o "Tesis de maestría".
-   `address`: La ubicación de la institución.
-   `month`: El mes en que se presentó la tesis.
-   `note`: Cualquier información adicional sobre la tesis.


## Ejemplo

He aquí un ejemplo de cómo utilizar el tipo BibTeX "phdthesis":

```bibtex
@phdthesis{mitesis,
	title = {Un análisis de ejemplo},
	author = {John Smith},
	year = 2022,
	month = {Junio},
	address = {Ciudad Ejemplo, CA},
	note = {Disponible en \url{https://example.com/thesis.pdf}},
	school = {Universidad de Ejemplo},
	type = {Tesis doctoral}
}
```



En este ejemplo, la entrada BibTeX define una tesis doctoral de John Smith titulada "An Analysis of Example". El título fue concedido en 2022 por la Universidad de Ejemplo, y la tesis fue presentada en junio en Example City, CA. El tipo de tesis se especifica como "Tesis doctoral", y se incluye una nota que proporciona una URL para la tesis.
