import CiteDrive from '@components/CiteDrive.astro';

BibTeX는 컴퓨터 과학 및 기타 기술 분야에서 널리 사용되는 참고 문헌 관리 도구입니다. BibTeX는 책, 기사, 학회 논문 등과 같은 참고 문헌을 구조화된 형식으로 저장하고 관리하는 데 사용됩니다. 이 형식은 다양한 문서 형식으로 자동으로 참고 문헌을 생성하는 데 사용될 수 있습니다.

<CiteDrive/>

BibTeX에는 다양한 출판물 유형을 나타내기 위해 각기 다른 필드를 가진 여러 가지 항목 유형이 있습니다. 다음은 모든 BibTeX 유형과 그 필드에 대한 종합적인 목록입니다:
