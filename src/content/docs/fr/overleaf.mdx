### BibTeX

BibTeX est un système de gestion des références largement utilisé dans LaTeX. Pour utiliser BibTeX dans Overleaf :

1. **Créer un Fichier de Bibliographie** : Créez un fichier nommé `references.bib` dans votre projet Overleaf.
2. **Ajouter des Références** : Incluez les références dans ce format :

   ```bibtex
   @article{AuthorYear,
       author = {Auteur, Prénom et Auteur, Deuxième},
       title = {Titre de l'Article},
       journal = {Nom du Journal},
       volume = {Numéro du Volume},
       number = {Numéro du Numéro},
       pages = {Numéros de Pages},
       year = {Année de Publication},
       publisher = {Nom de l'Éditeur}
   }
   ```

3. **Citer des Références** : Dans votre document LaTeX, ajoutez :

   ```latex
   \bibliography{references}
   \bibliographystyle{plain}
   ```

   Utilisez `\cite{AuthorYear}` pour citer une référence.

4. **Optimiser avec CiteDrive** : Automatisez et gérez vos références BibTeX plus efficacement avec CiteDrive.

### Natbib

Natbib offre une plus grande flexibilité pour le formatage des citations. Pour utiliser Natbib dans Overleaf :

1. **Ajouter des Commandes** : Incluez dans le préambule :

   ```latex
   \usepackage{natbib}
   \bibliographystyle{plainnat}
   ```

2. **Citer des Références** : Utilisez `\citep{AuthorYear}` pour les citations parenthétiques ou `\citet{AuthorYear}` pour les citations textuelles.

3. **Bibliographie** : Ajoutez :

   ```latex
   \bibliography{references}
   ```

4. **Améliorer avec CiteDrive** : Simplifiez la gestion et le formatage des citations avec CiteDrive.

### BibLaTeX

BibLaTeX fournit des fonctionnalités de citation avancées. Pour utiliser BibLaTeX dans Overleaf :

1. **Ajouter des Commandes** : Incluez dans le préambule :

   ```latex
   \usepackage[style=authoryear,backend=bibtex]{biblatex}
   \bibliography{references}
   ```

2. **Citer des Références** : Utilisez `\parencite{AuthorYear}` pour les citations parenthétiques ou `\textcite{Author}` pour les citations dans le texte.

3. **Optimiser avec CiteDrive** : Profitez d'un contrôle et d'une flexibilité accrus avec le support de CiteDrive pour BibLaTeX.

## Bases de LaTeX

Pour ceux qui découvrent LaTeX, voici un aperçu rapide :

```latex
\documentclass{article}
\usepackage{graphicx}
\usepackage{caption}
\usepackage{biblatex}

\addbibresource{references.bib} % Remplacez "references.bib" par votre fichier de bibliographie

\title{Mon Premier Document LaTeX}
\author{Votre Nom}

\begin{document}

\maketitle

\section{Introduction}
Ceci est un document d'exemple montrant les bases de LaTeX.

\section{Mise en Forme}
Vous pouvez formater le texte en \textbf{gras}, \textit{italique}, \underline{souligné}, ou en \texttt{police à chasse fixe}.

\section{Listes}
Voici un exemple de liste à puces :
\begin{itemize}
  \item Premier élément
  \item Deuxième élément
  \item Troisième élément
\end{itemize}

\section{Mathématiques}
LaTeX est excellent pour la composition de formules mathématiques. Voici un exemple d'équation :
\begin{equation}
  E = mc^2
\end{equation}

\section{Figures et Tableaux}
Vous pouvez inclure des figures et des tableaux dans votre document. Voici un exemple de figure :
\begin{figure}[ht]
  \centering
  \includegraphics[width=0.5\textwidth]{example-image}
  \caption{Une figure exemple}
  \label{fig:example}
\end{figure}

Et voici un exemple de tableau :
\begin{table}[ht]
  \centering
  \begin{tabular}{|c|c|}
    \hline
    \textbf{Élément} & \textbf{Quantité} \\
    \hline
    Pomme & 3 \\
    Orange & 5 \\
    \hline
  \end{tabular}
  \caption{Un tableau exemple}
  \label{tab:example}
\end{table}

\section{Références}
Vous pouvez faire référence à des sections, équations, figures et tableaux étiquetés. Par exemple, voir la Figure~\ref{fig:example} et le Tableau~\ref{tab:example}.

\section{Citations et Bibliographie}
Citez des références en utilisant \cite{author2022}. Voici un exemple de citation.

\printbibliography

\end{document}
```
