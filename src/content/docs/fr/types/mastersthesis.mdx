BibTeX est un outil de gestion des références couramment utilisé dans les documents LaTeX. Le type BibTeX "mastersthesis" est utilisé pour les mémoires de master. Dans ce guide, nous expliquerons les champs requis et optionnels pour le type BibTeX "mastersthesis".

<CiteDrive/>

## Champs Requis

Le type BibTeX "mastersthesis" nécessite les champs suivants :

-   `author` : L'auteur du mémoire.
-   `title` : Le titre du mémoire.
-   `school` : Le nom de l'institution qui a délivré le diplôme.
-   `year` : L'année où le diplôme a été délivré.

## Champs Optionnels

En plus des champs requis, le type BibTeX "mastersthesis" comporte plusieurs champs optionnels qui peuvent être utilisés pour fournir des informations supplémentaires. Ces champs incluent :

-   `type` : Le type du mémoire, comme "Mémoire de Master".
-   `address` : Le lieu de l'institution.
-   `month` : Le mois où le mémoire a été soumis.
-   `note` : Toute information supplémentaire concernant le mémoire.

## Exemple

Voici un exemple de l'utilisation du type BibTeX "mastersthesis" :

```bibtex
@mastersthesis{mythesis,
	title        = {Une Étude d'Exemple},
	author       = {Jane Doe},
	year         = 2022,
	month        = {Juin},
	address      = {Example City, CA},
	note         = {Disponible à \url{https://example.com/thesis.pdf}},
	school       = {Université d'Example},
	type         = {Mémoire de Master}
}
```

Dans cet exemple, l'entrée BibTeX définit un mémoire de master rédigé par Jane Doe intitulé "Une Étude d'Exemple". Le diplôme a été délivré en 2022 par l'Université d'Example, et le mémoire a été soumis en juin à Example City, CA. Le type du mémoire est spécifié comme "Mémoire de Master", et une note est incluse fournissant une URL pour le mémoire.
