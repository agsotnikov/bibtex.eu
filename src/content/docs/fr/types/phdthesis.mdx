BibTeX est un outil de gestion des références couramment utilisé dans les documents LaTeX. Le type d'entrée "phdthesis" dans BibTeX est utilisé pour les thèses de doctorat ou les dissertations. Dans ce guide, nous expliquerons les champs requis et optionnels pour le type d'entrée "phdthesis".

<CiteDrive/>

## Champs Requis

Le type d'entrée "phdthesis" dans BibTeX nécessite les champs suivants :

-   `author` : L'auteur de la thèse.
-   `title` : Le titre de la thèse.
-   `school` : Le nom de l'institution qui a décerné le diplôme.
-   `year` : L'année où le diplôme a été décerné.

## Champs Optionnels

En plus des champs requis, le type d'entrée "phdthesis" dans BibTeX possède également plusieurs champs optionnels qui peuvent être utilisés pour fournir des informations supplémentaires. Ces champs incluent :

-   `type` : Le type de la thèse, tel que "Thèse de doctorat" ou "Mémoire de Master".
-   `address` : Le lieu de l'institution.
-   `month` : Le mois où la thèse a été soumise.
-   `note` : Toute information supplémentaire sur la thèse.

## Exemple

Voici un exemple de comment utiliser le type d'entrée "phdthesis" dans BibTeX :

```bibtex
@phdthesis{mythesis,
	title        = {Une Analyse de l'Exemple},
	author       = {John Smith},
	year         = 2022,
	month        = {Juin},
	address      = {Example City, CA},
	note         = {Disponible à \url{https://example.com/thesis.pdf}},
	school       = {Université de l'Exemple},
	type         = {Thèse de doctorat}
}
```

Dans cet exemple, l'entrée BibTeX définit une thèse de doctorat rédigée par John Smith intitulée "Une Analyse de l'Exemple". Le diplôme a été décerné en 2022 par l'Université de l'Exemple, et la thèse a été soumise en juin à Example City, CA. Le type de la thèse est précisé comme "Thèse de doctorat", et une note est incluse fournissant un URL pour la thèse.
