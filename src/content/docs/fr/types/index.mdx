BibTeX est un outil de gestion de références bibliographiques populaire utilisé dans le domaine de l'informatique et d'autres disciplines techniques. Il est utilisé pour stocker et gérer des références bibliographiques, telles que des livres, des articles et des communications lors de conférences, dans un format structuré. Ce format peut ensuite être utilisé pour générer automatiquement des bibliographies dans divers formats de documents.

<CiteDrive />

BibTeX propose plusieurs types d'entrées, chacun avec son propre ensemble de champs, pour représenter différents types de publications. Voici une liste complète de tous les types BibTeX et de leurs champs :
