## Créer un fichier bib à la main

Créer un fichier `.BIB` est en réalité très simple. Il vous suffit de créer un fichier texte avec l'extension `.bib`. Vous pouvez ensuite ajouter vos références BibTeX à ce fichier, une par ligne. Par exemple :

```latex
	@article{smith201X,
		title        = {Un article intéressant},
		author       = {John Smith},
		year         = {201X},
		journal      = {Journal des Articles Intéressants}
	}
```

Une fois que vous avez créé votre fichier `.BIB`, vous pouvez l'utiliser dans vos documents LaTeX en spécifiant la commande `\bibliography{filename}`. Par exemple :

```latex
\documentclass{article}
\usepackage{natbib}

\begin{document}

...

\bibliography{filename}

\end{document}
```

C'est tout ! Créer un fichier `.BIB` est un moyen simple de gérer vos références BibTeX. N'oubliez pas de consulter nos autres articles sur BibTeX pour plus de conseils et astuces !
