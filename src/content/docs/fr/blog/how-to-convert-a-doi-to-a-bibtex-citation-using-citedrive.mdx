Si vous êtes chercheur, vous savez à quel point les bibliographies et les citations sont importantes. Et si vous utilisez [CiteDrive](https://www.citedrive.com/), vous savez que c'est le moyen le plus simple de créer des bibliographies et des citations pour vos articles de recherche. Mais que faire si vous tombez sur un [DOI](/fields/doi/) au lieu d'un ISBN ou d'une URL ? Pas de souci – nous avons ce qu'il vous faut ! Dans cet article de blog, nous allons vous montrer comment convertir un DOI en citation BibTeX avec CiteDrive. C'est parti !

## Utilisation de l'extension de navigateur  

Convertir un DOI en citation BibTeX est facile si vous utilisez l'extension de navigateur 'CiteDrive Companion'. Il vous suffit de surligner le DOI avec la souris, d'ouvrir le menu contextuel avec le bouton droit et de sélectionner 'Rechercher DOI'.

![CiteDrive - Exemple](@assets/images/tutorial/browserextension-search-doi.png)

Ensuite, vous verrez un aperçu de l'entrée BibTeX où vous pouvez la modifier avant de l'enregistrer dans un projet de votre choix.

![CiteDrive - Exemple](@assets/images/tutorial/browserextension-bibtex-preview.png)

C'est tout ! Vous trouverez l'extension compagnon disponible pour les navigateurs suivants :

* [CiteDrive Companion Plugin pour Google Chrome](https://chrome.google.com/webstore/detail/citedrive-bibtex-companio/gmmonfphegngpcbcapfbgembkjeookik)  
* [CiteDrive Companion Plugin pour Mozilla Firefox](https://addons.mozilla.org/af/firefox/addon/citedrive-companion/)  
* [CiteDrive Companion Plugin pour Microsoft Edge](https://microsoftedge.microsoft.com/addons/detail/citedrive-bibtex-compan/kpmepfpbjehnlgfbcnbmgihklkkmfnpo)

## Utilisation de l'outil en ligne :  
Si vous n'utilisez pas l'extension de navigateur, pas de souci – vous pouvez toujours utiliser CiteDrive pour convertir votre DOI en citation BibTeX. Il vous suffit d'aller sur le site de CiteDrive et de coller votre DOI dans le champ 'Entrer DOI', que vous trouverez sous "Rechercher via DOI" en cliquant sur le symbole plus dans la barre de navigation supérieure. Cliquez ensuite sur le bouton 'Convertir', et voilà ! – votre citation BibTeX sera générée automatiquement.

![CiteDrive - Exemple](@assets/images/tutorial/citedrive-search-doi.png)
