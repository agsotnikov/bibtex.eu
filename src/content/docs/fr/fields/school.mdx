Dans BibTeX, le champ `school` est principalement utilisé dans les entrées `@phdthesis` ou `@mastersthesis`. Ce champ est utilisé pour indiquer l'institution éducative où la thèse ou le mémoire a été rédigé. C'est un champ obligatoire pour ces deux types d'entrée, fournissant des informations détaillées sur l'origine du travail académique, ce qui peut être important pour les lecteurs qui souhaitent obtenir plus d'informations ou référencer le travail dans leurs propres recherches.

## Directives de Format

Lors de l'entrée du nom de l'école, il est conseillé d'utiliser le nom complet et officiel de l'institution pour maintenir un format formel et standardisé dans vos références. Évitez d'utiliser des abréviations sauf si elles sont largement reconnues et acceptées.

## Exemples

Voici des exemples de l'utilisation du champ `school` pour les entrées `phdthesis` et `mastersthesis` :

### Thèse de Doctorat

```bibtex
@phdthesis{doe2023,
  author = {John Doe},
  title = {Explorations en Informatique Quantique},
  year = {2023},
  school = {Massachusetts Institute of Technology},
}
```

### Mémoire de Maîtrise

```bibtex
@mastersthesis{smith2023,
  author = {Sarah Smith},
  title = {Étude Complète sur la Durabilité Environnementale},
  year = {2023},
  school = {University of California, Berkeley},
}
```

## Remarque

Dans votre exemple, vous avez utilisé le champ `series` au lieu du champ `school` pour l'entrée `mastersthesis`. Cependant, le champ `school` doit être utilisé pour spécifier l'institution dans les entrées `phdthesis` et `mastersthesis`, comme le montrent les exemples ci-dessus.
