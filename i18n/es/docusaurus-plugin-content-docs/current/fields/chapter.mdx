## BibTeX field type: chapter

:::note
Esta página sólo está disponible en inglés, pero necesitamos tu ayuda para traducirla a tu idioma. Si estás interesado, consulta nuestro repositorio de GitHub para obtener más información sobre cómo contribuir.
:::

A chapter, section, or a range of pages of a book is stored numerically in ``chapter``.
This BibTeX-field is used for ``inbook`` and ``incollection``. The former is mandatory.
``chapter`` is not supported in the entry-type ``book`` because for the latter, the whole book is intended to be cited.

## Format

```tex
@book{Mueller2014The,

  ...

  chapter={ 2 }

  ...

}
```
