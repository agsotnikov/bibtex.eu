# BibTeX field type: school

:::note
Esta página sólo está disponible en inglés, pero necesitamos tu ayuda para traducirla a tu idioma. Si estás interesado, consulta nuestro repositorio de GitHub para obtener más información sobre cómo contribuir.
:::

`school` is specified when a `phdthesis` or `mastersthesis` is cited and is the educational institution where the student wrote the dissertation. This field is required in both entry types.

```tex
@phdthesis{key,
  school={University of ...},
}


@mastersthesis{key,
  series={University of ...},
}
```
